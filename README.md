# 「花瓣数据图」Android 客户端

读图时代，从「花瓣」上阅读每日更新的数据图。


## FAQ

### 为什么大图情况下无法显示图片？

为了节省运营商流量，大图只允许在 Wifi 情况下访问，但您扔可以点击载入大图（这可能会耗费您多余的流量）。

### 有些图片似乎不清晰，怎么办？

默认情况下应用设置为「使用高清图片」，但或许您关闭了这个选项，您可以在「设置」中检查是否开启了这个选项。如果在开启了这个选项的情况下还是模糊，那么抱歉可能数据源提供的图片本身就是模糊的。
